Welcome to UT SSL
=================
This is the basic structure of a web app project with UT and SSL
Ready for use !

Esta es la estructura básica de un proyecto app web con UT y SSL 

------------------------------------------------------------------------------

Tienes que generar tus certificados, sino la aplicación no funcionará.
Busca mas información en https://carles9000.github.io/ y busca SSL

https://carles9000.github.io/?search=ssl


You have to generate your certificates, otherwise the application will not work
Look for more information at https://carles9000.github.io/ and look for SSL

https://carles9000.github.io/?search=ssl 

------------------------------------------------------------------------------

En la carpeta \dll tienes las diferentes dlls que necesitaras si no las tienes 
instaladas en tu windows\system32. Si las instalas en el directorio del exe 
tambien se ejecutará correctamente.

In the \dll folder you have the different dlls that you will need if you don't have 
them installed on your windows\system32. If you install them in the exe directory 
they will also will execute correctly.
